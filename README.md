# Task-Manager
# 🧠 Task Manager Web App



---

## 🚀 Clone & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/prasad-dotcom/Task-Manager.git
cd Task-Manager
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm run dev
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```

---

## 🚀 Features Implemented

### 🔐 Authentication

* User Signup & Login
* JWT-based authentication
* Password hashing using bcrypt
* Protected routes

---

### 📋 Task Management

* Create tasks
* View all tasks
* Update tasks
* Delete tasks
* Mark task as completed

---

### 📝 Task Fields

* Title
* Description
* Status (Todo / In Progress / Done)
* Priority (Low / Medium / High)
* Due Date

---

### 🔍 Filtering & Search

* Filter by status
* Filter by priority
* Search tasks by title

---

### 📊 Analytics

* Total number of tasks
* Number of completed tasks
* Number of pending tasks
* Completion percentage

---

### ✨ Enhancements

* Pagination on task list
* Sorting (due date / priority)
* Responsive design
* Clean UI
